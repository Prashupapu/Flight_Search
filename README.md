How to run application
Step1 : Go to start button in visual studio->Select Configure Startup Projects->select Multiple start Projects->action is **start** for flight_search.client
->action is **start** for flight_search.server
Step2 : Install->https://nodejs.org/en/
Step3 : Go to Tools->NuGet Package Manager->Package Manager Console->npm install axios@latest under flight_search.client.
Step4: Command promte for UI press h + enter->o + enter
